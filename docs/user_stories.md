# 📋 User Stories for Library Management System

## 🧑‍💻 General User Stories

1. **As a user**, I want to borrow a book so that I can read it at home.
2. **As a user**, I want to return a borrowed book so that it becomes available to others.
3. **As a user**, I want to view my borrowed books so that I can keep track of what I need to return.

---

## 🧑‍💼 Librarian User Stories

1. **As a librarian**, I also want to be able to borrow and return books, but without a limit on how many I can borrow at once.
